# Encephalitis International Branding Assets

## Logo Acquisition

To obtain the official Encephalitis International logo:

1. Visit [Encephalitis International](https://www.encephalitis.info/)
2. Contact their team to request official logo files:
   - Email: support@encephalitis.info
   - Phone: +44 (0) 1653 699 599

3. Request the following formats:
   - **SVG format** (preferred for web - scalable)
   - **PNG format** with transparent background (fallback)
   - Both light and dark versions if available

4. Place the files in this directory:
   - `encephalitis-international-logo.svg`
   - `encephalitis-international-logo.png`
   - `encephalitis-international-logo-dark.svg` (if available)

## Brand Guidelines

Please ensure you follow Encephalitis International's brand guidelines when using their logo:
- Maintain proper spacing around the logo
- Do not alter colors or proportions
- Include proper attribution
- Link logo back to https://www.encephalitis.info/

## Tagline

**"The brain inflammation charity"**

This tagline should accompany the logo in the header.

## Partner Badges

If you have access to partner certification badges (Lottery Funded, Carbon Neutral, etc.), place them in the `partner-badges/` subdirectory.

## Temporary Placeholder

Until you obtain the official logo, the application will use a text-based placeholder that reads "Encephalitis International".
