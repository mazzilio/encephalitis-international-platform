# Monitoring Tools

This directory contains monitoring tools for tracking processing progress in real-time.

## Scripts

### monitoring_server_enhanced.py
Enhanced monitoring server with real-time dashboard.

**Features**:
- Live progress tracking
- Processing statistics
- Confidence score distribution
- Resource type breakdown
- Real-time charts and graphs

**Usage**:
```bash
# Start monitoring server
python3 monitoring_server_enhanced.py

# Open browser
http://localhost:5000
```

### test_monitoring_demo.py
Demo script for testing the monitoring system.
